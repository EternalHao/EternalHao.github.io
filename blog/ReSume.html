                  # 本篇教程为楼主毕生吐血所做,为的就是讨好面试官,让你脱颖而出,找到你理想的工作

                  WARNING

                  本套教程适合你面试 (100-1000人) 以内的公司,不适合大厂,尤其是喜欢用网上问烂的面试题,教你搞定面试官

                  # 写简历

                  这个问题,对于我来说很难,但是总是要面对的,我的简历已经经过 1,2,3...版后了,但是对于现在的这个版本还是不太满意,简历需要美化、简历需要美化、简历需要美化、,你要知道大家做的事情其实都是查不多的,不要认为你做的很平凡,平凡就是一种伟大,你做的额事情一定有亮点,试着将它表达出来,然后再加以美化. 简历是很重要的,完全取决你有没有面试的资格

                  # 接电话

                  时代在进步,社会在发展,如果你已经很久没有出来面试下,免的被社会淘汰,大胆尝试,证明自己,我们一起共同成长,揭秘面试中的套路,拿到你喜欢的Offer(如果你老实,就是一个憨憨)

                  一般来说在你简历投递以后,如果公司有意向,在你投简历的1~2天后,就会打电话联系你

                  第一次的电话沟通很重要,一般讨论点都在下面的几个方面

                  # 1. 你期待的薪资?

                  回答这个问题,不要犹豫,直接说出你的理想薪资,这时候就有两种情况

                  • a. 你的薪资是一个区间
                  • b. 你的薪资是一个具体的值

                  这两个有什么区别呢,区别就在于你后来能得到多少,换句话说你的面试结果会决定你最后谈薪资 首先看你的面试的结果:

                  • a. 不合格
                  • b. 合格
                  • c. 超出预期
                  • d. 特殊原因 面试的结果无非就是这四种,其中b,c,d都是公司对你有想法,换句话说你通过面试了

                  那我们来看下一 你的面试结果 -- 薪资之间的关系

                  对于你提出的 a中薪资的算法:

                  • b. -- a区间的最低值
                  • c. -- a的中间值(少数也有可能达到最大值)
                  • d. -- 由于是特殊原因 (结果我觉得是最小值,中间值,少数情况是最大值)

                  对于你提出的 b中薪资的算法:

                  • b. -- 坑定要压你的工资(如果你不接收,那你可能是个憨憨)
                  • c. -- 就是你要的工资
                  • d. -- 就是你要的工资

                  TIP

                  对于这两种提工资的方法都没有问题,问题就在你最终能得到多少(不要钻牛角尖,我说的是大多数的情况),至于工资怎么定,全看自己了,但是想说的一点是,你的工资可以适当高点,有利于讨价还价,这样在最终环节可能才会拿到你理想的工资(人都是喜欢占便宜的,公司也一样).

                  # 2. 你为什么离职?

                  这个问题,说难也难,说不难也不难,只要保证一点就是 不要抱怨公司, 利用一些地理位置转化去说,“比如就说我搬家了,离现在的公司比较远,所以想换个环境,找一个非客观的理由.

                  # 3. 什么时间可以面试(电话面试)?

                  这个由于我是在职,我一般晚上有时间,所以大概率是晚上,这个你自己看时间定

                  # 确认公司背景

                  公司背景及法人信息

                  营业执照

                  # 确认路线,尽量不要迟到,一般提前10-20分钟到公司

                  为什么要提前到公司呢,主要有两点

                  1. 提前到达面试现场,我认为这个是必要的
                  2. 你可以提前看看公司环境,这毕竟有可能是你日后工作的地方,一定要好好了解一下的

                  # 第一步 填表格

                  经历了上面的,人事可能就已经找到你了,大概率会让你填一份表格,入职登记表,表格里的信息,大概有以下几种

                  1. 你的基础信息 姓名,年龄,性别,政治身份,现住址,电话,毕业院校及时间
                  2. 你的家庭信息 (这个可以不填 即使填也不要填写家庭任何成员的电话)
                  3. 你的任职信息 (这个如实填写,这个大概率会回访)

                  最重要的一点就是,在最下面有你的签名,上面有一行小字“本人保证所填写的信息真实准确,如有虚假,可以终止劳动合同”,这个可能是公司会日后用来和你撕逼的重要利器(逼你走的绝招),所以尽量保证真实.

                  # 第一轮 面试

                  第一轮面试,基本都是基础性的问题,时间一般在 1个小时左右,也就是我们精心准备的,尽量衍生话题,让面试官跟着自己走,让他少问问题,

                  # 第二轮 面试

                  第一轮面试结束后,如果有机会,面试官会让你等一下下一个面试官,第二轮的面试的侧重点,应该是考察你的项目,换句话说就是这个项目是不是你做的,还是你没做过写在简历上的,我们也准备到了应对措施,看项目那一节

                  # 第三轮 面试

                  经历了前两轮的面试,你可以提出上个厕所,随便恢复一下心情,观察一下环境,准备下人事面,人事面的目的就是一个就是你这个人价值观有没有问题,适不适合公司, 一般都是一些常见的问题,主要包括生活上的问题和你在职场上一些遭遇

                  这个是最考验说话技巧的, 问题也都是比较刁钻的.后续会整理一份文档

                  # 等通知

                  说实话,这个等通知是面试中最骚的,解决这个的问题

                  1. 和你的人事关系处的稍微好一点,直接私聊问他
                  2. 真的傻傻等通知
